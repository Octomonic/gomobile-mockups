0:42–0:54Making a price change, end-to-endביצוע שינוי מחיר, מקצה לקצה

Do one live, start to finish, on the agreed test SKU:בצעו אחד חי, מתחילה לסוף, על מק״ט הבדיקה המוסכם:

  1. ⟳ Refreshרעננו the row first — pull the latest price / stock / GoSale before deciding.קודם את השורה — משכו מחיר / מלאי / GoSale עדכניים לפני שמחליטים.
  2. Open the price-change panel → enter regular + Eilat price (the panel computes the warranty/insurance fields).פתחו את חלונית שינוי המחיר ← הזינו מחיר רגיל + אילת (החלונית מחשבת את שדות האחריות/ביטוח).
  3. If the colours sit at different prices → the variant-decision popup appears. The 3 choices:אם הצבעים במחירים שונים ← מופיע חלון החלטת הווריאנט. 3 הבחירות:
    • לאחד / unify — all colours to one price (the usual choice).כל הצבעים למחיר אחד (הבחירה הרגילה).
    • shift — move all colours by the same ₪ (keeps intentional differences).להזיז את כל הצבעים באותו ₪ (שומר הבדלים מכוונים).
    • per-variant — set each colour separately (rare; e.g. Suunto titanium colours genuinely differ).לקבוע כל צבע בנפרד (נדיר; למשל צבעי טיטניום של Suunto באמת שונים).
  4. Pushדחפו → watch the mini-log:← צפו במיני-לוג: מאושר (doneבוצע) · ממתין (waitהמתינו) · הוחל על N מתוך M (only some colours took → clickרק חלק מהצבעים נקלטו ← לחצו ↻ ניסיון חוזר).
  5. GoSale-locked SKU:מק״ט נעול ב-GoSale: show the block + the override checkbox.הציגו את החסימה + תיבת העקיפה.
Your turn (sandbox):תורכם (סביבת בדיקה): push a tiny change and read the verify result. Use the agreed test SKU only.דחפו שינוי קטן וקראו את תוצאת האימות. השתמשו רק במק״ט הבדיקה המוסכם.
0:54–1:00Do's, Don'ts & Q&Aמותר, אסור ושאלות

✅ Doמותר

  • Refresh before judging.לרענן לפני שיפוט.
  • Respect importer tiers.לכבד דרגות יבואן.
  • Read severity before chasing a gap.לקרוא חומרה לפני שרודפים אחרי פער.
  • Confirm a push landed in the mini-log — "sent OK" ≠ "applied".לוודא שדחיפה נחתה במיני-לוג — "נשלח" ≠ "הוחל".

⛔ Don'tאסור

  • Price-match a phantom / contact-only competitor.להתיישר לפי מתחרה פנטום / צרו-קשר-בלבד.
  • Override a GoSale lock without a reason.לעקוף נעילת GoSale ללא סיבה.
  • Unify colours that are meant to differ.לאחד צבעים שאמורים להיות שונים.

Escalate to Tom:העבירו לתום: a >10% gap you can't explain · a push still stuck after retry · a competitor that looks mismatched but isn't blocked.פער מעל 10% שאי אפשר להסביר · דחיפה שעדיין תקועה אחרי ניסיון חוזר · מתחרה שנראה לא-תואם אבל לא חסום.
